# EvenDex

# How to use

    # Go

        inside 'src' folder

        go build views.go model.go indexer.go

        ./views /path/to/config/dev.conf

        go api hosted: http://localhost:8000/api/v1/events
    
    # Flask

        inside 'flask' folder

        python views.py

        flask frontend hosted: http://localhost:5000/events
